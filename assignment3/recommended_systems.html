Valantis Sector Map Valantis Commission Board Recommended Systems
Source Code (Github)

This can be used with an assortment of TTRPGs. It's main purpose is to procedually generate "cities" and commissions to give DM's and players something to play with. This specific version with flavored text is based on my own world building for a homebrew of Dungeons and Dragons 5e campaign which is set in a more modern world where the 4 elemental planes have converged into the material plane, so there is a lot of elemental influences. But this can be changed to include other influences in the source code.
